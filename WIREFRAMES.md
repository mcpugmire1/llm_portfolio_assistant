## Key Responsive Patterns Summary

### Navigation
- **iPhone SE/14:** Hamburger menu (â˜°)
- **iPad:** Horizontal tabs

### Avatars
- **iPhone SE:** 40-48px (conversation), 80px (hero)
- **iPhone 14:** 44-52px (conversation), 96px (hero)
- **iPad:** 56-60px (conversation), 120px (hero)

### Typography
- **iPhone SE:** 14px body, 18-24px headings
- **iPhone 14:** 15px body, 20-26px headings
- **iPad:** 16px body, 24-32px headings

### Layouts
- **iPhone SE/14:** Single column, stacked
- **iPad:** Multi-column grids (2-3 columns)

### Touch Targets
- **iPhone SE:** 44px minimum
- **iPhone 14:** 48px recommended
- **iPad:** 44-48px (desktop-like)

### Padding/Spacing
- **iPhone SE:** 12px messages, 16px containers
- **iPhone 14:** 14px messages, 20px containers
- **iPad:** 20px messages, 24px containers
